# 📖 Project Overview

This project demonstrates the design and implementation of a scalable **University Campus Network Infrastructure** using Cisco Packet Tracer.

The network consists of a **Main University Campus**, a **Branch Campus**, and **Cloud Connectivity** with complete VLAN segmentation, inter-VLAN routing, routing between campuses, and centralized services.

The project follows enterprise networking concepts commonly used in real-world organizations.
